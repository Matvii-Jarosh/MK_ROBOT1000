MK_ROBOT1000 - Moderator Keeper ROBOT1000

Це алгорітм знаходження поганих слів в реченні та додаткові функції для очищення текста.


Оснвні функції:

def contains_bad_word(word: str) -> tuple[bool, str | None, int | None]

Перевіряє на погані слова

Params:

word – слово на перевірку

Returns:

Чи є слово поганим, і за чого воно погане, ступінь поганості слова


def check_sentence_for_bad_words(sentence: str) -> list[tuple[str, str, int]]

Перевіряє речення на погані слова

Params:

sentence – речення яке перевіряємо

Returns:

список усіх поганих слів де на початку саме слова, аналог в словарі та ступінь поганості слова


def replace_bad_words(sentence: str) -> str

Замінює усі погані слова в реченні на зірочки

Params:

sentence – рядок з поганими словами

Returns:

рядок з заміненими поганими словами


Приклади:

result, matched_word, severity = contains_bad_word("п@р@ша")

print(result) # True

bad_words = check_sentence_for_bad_words("Ти 7упий, та ти l0$3r")

print(bad_words) # [('7упий', 'туп', 1), ('l0$3r', 'loser', 1)]


new_line = replace_bad_words("Ти туп0й!")

print(new_line) # Ти *****!
